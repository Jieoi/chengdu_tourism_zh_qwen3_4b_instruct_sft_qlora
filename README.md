<h1>QLoRA Fine-Tuning of Qwen3-4B on a Curated Chinese Tourism Q&A Corpus from Undergraduate Contributors</h1>
This repository contains the training, augmentation, and evaluation pipeline for a lightweight Chinese tourism assistant built under severe data scarcity using controlled style augmentation, QLoRA fine-tuning, and multi-stage evaluation.

Please see my medium Article on this project at: https://medium.com/@xinnnjie.li/building-a-domain-specific-chinese-tourism-assistant-from-data-scarcity-to-fine-tuning-685ac7832f79?postPublishedType=initial
